# My-App-Contact
Provides the backend framework for the contact form of danielyzhang.com with Node.js

### To run server:
```
nodemon index.js
```

### To push updates to GitHub:
````
git add .
git commit -m "Add existing file"
git push origin
````

## To push to Heroku:
```
git add .
git commit -am "make it better"
git push heroku
```
